# YourOS
Made with love!

I made YourOS with the help of Jehovah!

YourOS is a OS Builder with Alpine Linux Base.

### Requirements

1. QEMU System X86 84

2. cx (https://github.com/zer0users/cx)

3. Lua

4. VirtIO (For QEMU)

### How to compile

Add an disk.qcow2 With Alpine Linux.

Run:

```bash
cx build
```

On YourOS folder.

It may return an YourOS.cxA


### How to run

Run:

```bash
cx YourOS.cxA
```
